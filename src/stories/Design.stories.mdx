# Design

Pour des raisons de simplicité le design se base sur [material-ui](https://material-ui.com/). Il s'agit d'une implémentation en React du design system de Google, material.  
Les [guidelines de material](https://material.io/design/) sont donc applicables.

## Thème

Toute l'application React doit être contenu dans le composant `DvcThemeProvider` qui définit le thème material-ui. Voir [ThemeProvider](https://material-ui.com/customization/theming/#theme-provider).

```jsx
import React, { FC } from 'react';
import { DvcThemeProvider } from 'dvc-components';

const App: FC = () => <DvcThemeProvider>
    <>Le reste de l'application...</>
</DvcThemeProvider>;
```

## Typographie

La police à utiliser est Ubuntu : https://fonts.google.com/specimen/Ubuntu.  
L'échelle typograpgique est celle de material : https://material.io/design/typography/the-type-system.html#type-scale.  

Le composant [Typography](https://material-ui.com/components/typography/) de material-ui permet de respecter cette échelle dans la partie React.

## Couleurs

La palette se décline en 12 couleurs, dont les deux premières sont la couleur primaire, et la couleur secondaire.

### Utilisation

Il faut privilégier l'utilisation des variables css au code hexadécimal, si possible.  
Le css permettant de déclarer ces variables est accessible [plus bas](#variables-css).

```css
button {
    /* Priviliégiez les variables */
    color: var(--clr-primary);
    /* Plutôt que le code hexa */
    color: #005fb8;
}
```
