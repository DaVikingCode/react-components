# DVC Components

Librairie de composant et documentation pour le développement d'application React chez DVC.

## Dépendances

- Cartographie : https://github.com/DaVikingCode/react-openlayers/
- Composants : https://material-ui.com/
- Style : https://styled-components.com/
- State Management : https://mobx.js.org/

### Cartographie

Les abstractions relatives à la cartographie se trouve dans le repo : https://github.com/DaVikingCode/react-openlayers, à utiliser conjointement à cette librairie pour afficher une carte.
